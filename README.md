# Flask + Node.js Docker Assignment

## 📌 Description

This Assignment demonstrates a full-stack application using:

* Flask (Backend)
* Node.js + Express (Frontend)
* Docker & Docker Compose

## 🚀 How to Run

```bash
docker-compose up --build
```

## 🌐 Access

* Frontend: http://localhost:3000
* Backend: http://localhost:5000

## 🐳 Docker Hub Images

* Backend: [Backend Repository Link](https://hub.docker.com/repository/docker/hanishabogadhi/backend/general)
* Frontend: [Frontend Repository Link](https://hub.docker.com/repository/docker/hanishabogadhi/frontend/general)

## 📂 Project Structure

backend/ → Flask app
frontend/ → Node app
docker-compose.yml → Service orchestration
